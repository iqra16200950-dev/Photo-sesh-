# Photo-sesh-
WELCOME to the SweetSnaps photobooth!!!

overview:
This project is a web base photobooth application made by using react.I really wanted something to resonate and recreate the experiance of taking photobooth style pictures in a simple and interactive way . Another reason this was made is due to the fact that i have long distnce friends - meaning we dont get to take photos together ,and even though i still cant magically spawn them , its still really fun for me to see or send pictures :).

HOW THIS WORKS :
Users can select from 4 frames (which were drawn by me) .They can capture or upload pictures and place them in the structured photostrip.This can also be customized using the stickers present .

HOW IT WAS BUILT 
This project was developed using HTML,CSS but mainly JAVASCRIPT and React .It consists of camera integration ,a timer system,image handling,dynamic rendering and a download feature.
